# cbir-systeam
一个本地的基于内容的图像检索系统，实现了包括颜色特征提取（颜色直方图，HSV中心距），纹理特征（灰度共生矩阵，LBP算子），边缘特征（边缘直方图），哈希感知算法（aHash,pHash,dHash算法等），SIFT特征提取。以及基于VGG-16提取特征等功能
